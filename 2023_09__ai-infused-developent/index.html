What is a language model?

Large language models are advanced AI systems that are capable of understanding and generating human language.

They are built using complex neural network architectures, such as transformer models, inspired by the human brain.

How do I know which model or prompt to turn to?

Chat GPT-3 & GPT-4

  • 👍 Better at writing
  • 👍 Has more mature responses
  • 👍 Has more integrations/plugins
  • 👍 a collaborative experience
  • ❌ data source cut-off at 2021
  • ❌ not immune to hallucinations

Bard

  • 👍 Better for research
  • 👍 Better user experience
  • 👍 Draws from Google Search
  • ❌ Prone to hallucinations
  • ❌ Does not track previous requests

How do I know which Code Generator to turn to?

Github Copilot

  • 👍 Does single or multiline auto-completes
  • 👍 Integrated in IDE
  • 👍 Multiple commands can be given in a Chat
  • ❌ Paid

Amazon Code whisperer

  • 👍 Does single or multiline auto-completes
  • 👍 Integrated in IDE
  • ❌ No Inline Chat Available
  • 👍 Free

Google Bard

  • 👍 Decent code explenation
  • 👍 Up-to-date with latest frameworks
  • ❌ Not Integrated in IDE (yet)
  • ❌ No Inline Chat Available
  • 👍 Free
