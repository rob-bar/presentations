Ocelot

What is it?

  • Theming toolbelt
  • Mainly for Drupal 7
  • 3 different kinds (master, omega & outside drupal)
  • Decent & tested standards
  • Ready for project kick-off

Strengths and weaknesses

  • Structure
  • Project isolated not developer/machine isolated
  • Good linting tools(scss & js)
  • Flexible

  • Compilation Speed
  • 2 package managers (gem & npm)
  • No plugable components

Now

  • Current stable version 0.1.2
  • WIP Milestone 0.1.3
  • Integrated in Projectify
  • Wiki & Doc can be found here

Future plans and milestones

Milestone 0.1.3
Structure & general improvements

  • Structure
  • Better Doc & wiki
  • Susy grid config and mixins
  • Bower integration & injection into info file

Milestone 0.1.4
Speed boosts and no more gems

  • Convert to libsass
  • Replace all the gems by npm modules
  • Replace compass with bourbon
  • Grunt or Gulp investigation

Milestone 0.1.5
Components, GPI and D8 standards

  • Make components plugable (Messages, tabs, menu' s, etc.)
  • Integrate D8 standards (backbone, eshinting)
  • Google page speed insights on workflow
  • Critical css
